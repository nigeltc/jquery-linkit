#linkit jQuery plugin#

**linkit** is a jQuery plugin to attach links to elements.

##Version##
0.2.0

##Usage##

    <script>
        $(document).ready( function () {
            $('span').linkit( {
                href: 'http://test.com',
                text: 'something new'
            });
        } );
    </script>

##Vendors##
jQuery - [http://jquery.com](http://jquery.com)
